Projeto de Previsão de Vendas na Rossmann

Nesse projeto, foi possível conduzir uma análise abrangente para prever as vendas diárias nas lojas da Rossmann, empregando técnicas avançadas de Machine Learning. O objetivo era criar um modelo robusto que considerasse variáveis como promoções, concorrência, feriados e sazonalidade, proporcionando aos gerentes de loja a capacidade de otimizar suas estratégias e escalas de trabalho.

O resultado final é um modelo de previsão baseado em XGBoost, implementado com sucesso em uma API no Telegram. Os usuários podem acessar previsões precisas de vendas para as próximas 6 semanas, permitindo uma gestão eficiente das operações das lojas.

As ferramentas utilizadas foram:

  • Python 3.9 e Scikit-learn
  • Bibliotecas como pandas, numpy, matplotlib, seaborn
  • Algoritmos de Regressão: Linear Regression, Lasso, Random Forest, XGBoost
  • Métricas de performance: R2, MSE, RMSE, MAE, MAPE E MAP
  • Machine Learning com XGBoost
  • API no Telegram para disponibilizar previsões de vendas específicas para cada loja

Projeto de Previsão de Preços para Anúncios no Airbnb - Rio de Janeiro

Nesse projeto foi possível explorar o universo do Airbnb no Rio de Janeiro, aplicando técnicas de Machine Learning para prever preços de anúncios. O objetivo era criar um modelo capaz de fornecer insights valiosos para anfitriões e locatários, permitindo uma tomada de decisão mais informada.

O modelo inicial, baseado em Extratrees, entregou resultados impressionantes. No entanto, para viabilizar o deploy do projeto, fiz uma transição estratégica para o XGBoost, ajustando a complexidade do modelo. Essa mudança, embora tenha impactado algumas métricas, foi essencial para tornar o projeto funcional e eficaz.

O produto final é um painel online intuitivo, hospedado na nuvem, acessível a qualquer dispositivo. Anfitriões e locatários podem explorar previsões de preços com base nas características dos imóveis, tornando o processo de precificação mais transparente e eficiente.

As ferramentas utilizadas foram:

  • Python 3.9 e Scikit-learn
  • Bibliotecas como pandas, numpy, matplotlib, seaborn
  • Algoritmos de Regressão: Random Forest, Linear Regression, ExtraTrees, XGBoost
  • Métricas de Performance: R², MAE, MAPE, RMSE
  • Streamlit para construção do painel online
